# YouTube Downloader

Минималистичный веб-сервис для скачивания видео с YouTube.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Откройте браузер на `http://localhost:3000`

## Использование

1. Вставьте ссылку на YouTube видео
2. Нажмите "Скачать"
3. Дождитесь загрузки файла
