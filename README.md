# YouTube Downloader

Минималистичный веб-сервис для скачивания видео с YouTube.

## Использование

1. Вставьте ссылку на YouTube видео
2. Нажмите "Скачать"
3. Дождитесь загрузки файла

## GitHub Pages

Демо доступно по адресу: https://darkfimoz.github.io/YouTube-Downloader/

## Технологии

- HTML/CSS/JavaScript
- Cobalt API
