# YouTube Downloader

Minimalist web service for downloading YouTube videos.

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

Open browser at `http://localhost:3000`

## How to use

1. Paste YouTube video link
2. Click "Download"
3. Wait for file download

## GitHub Pages

Demo available at: https://darkfimoz.github.io/YouTube-Downloader/
